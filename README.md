# Game Text Editor
Vibe coded translation tool dedicated to Gotouchi Tetsudou translation.

It needs only Python 3 to work.

Was tailored to work on files generated by this script which parses text from files unpacked from `common.pac -> messages.pac`:
https://github.com/masagrator/NXGameScripts/blob/main/Gotouchi%20Tetsudou/message_entry_text_dumper.py

Archiving this here because I got good results with it and it can be repurposed for other games with Claude's help.

Functionality:
- Supports loading and saving multiple files at once
- Resizable text windows
- PREVIEW window properly showing Ruby text
- Configurable shortcuts
- Configurable text font
- Syntax highlighting for BR, RUBY and VO tags, and C format specifiers
- Tested on both Windows and MacOS
